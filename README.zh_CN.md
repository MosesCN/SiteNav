# 一站式互联网导航

一个基于 [Vue3](https://vuejs.org) + [Element Plus](https://element-plus.org) 的静态响应式网址导航

# 数据配置

数据配置在 [api/data](./api/data/)，包含

- 左侧菜单
- 搜索可选项
- 主页面网页导航地址信息

# 如何使用

1. 安装 Node 和 NPM，具体细节请询问 ChatGPT；
2. 获取本源码，并进入到源码根目录

```shell
git clone git@github.com:MosesCN/SiteNav.git
```

3. 编译调试

```shell
npm install
npm run dev
```
